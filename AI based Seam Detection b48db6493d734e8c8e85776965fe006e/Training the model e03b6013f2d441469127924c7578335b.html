Prepare the Ingredients

Once you have the training data prepared on a PC, the next step is to arrange everything neatly under a root directory. So, on the machine you are going to train on , create a folder called omcare-seam-detection

mkdir omcare-seam-detection
cd omcare-seam-detection

Data Directory - data

Copy the entire data folder created at the end of step 2 inside the omcare-seam-detection directory. 

#From the PC
scp data.zip admin@training_server:/omcare-seam-detection
#On the remote server
cd omcare-seam-detection
unzip data.zip
#remove the zipped data
rm data.zip

Final Model Directory - model

This is where the final model will reside after training. Create a folder called model under omcare-seam-detection 

cd omcare-seam-detection
mkdir model

Intermediate Model Directory - temp

Tensorflow uses a directory to store checkpoints, intermdiate model weights while training. create a directory to store this information as well

cd omcare-seam-detection
mkdir temp

Login to Docker

docker login
#Enter username omcareai
#Enter Password
>login succeeded!

Start Training

We will be using docker for training the neural netwotk

cd omcare-seam-detection
docker run \
--gpus all \
--rm \
--network=host \
-v ~/omcare-seam-detection/data:/home/omcare/model_trainer/data \
-v ~/omcare-seam-detection/model:/home/omcare/model_trainer/fine_tuned_model \
-v ~/omcare-seam-detection/temp:/home/omcare/model_trainer/training \
omcareai/seam-detector-trainer:latest

Take out the trained .tflite model

You will find the new models under the model directory with the extention .tflite
